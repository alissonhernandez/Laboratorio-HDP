Ocultar elementos Se puede ocultar o mostrar los elementos llamando a la clase, si deseas ocultar un elemento ocultar y para mostrar los elementos ocultos solo se llama la clase mostrar ejemplo:
Este div esta oculto
En el ejemplo anterior el div no se logra ver, ya que esta oculto pero este sigue ocupando el espacio en la pagina puedes inspeccionar presionando F12 y comprobar que esta ahi. Codigo
<div class="rde-s p-5 b-negro-1 sf-4 m-5 cf-blanco ocultar">Este div esta oculto</div>
Mostrar elementos ocultos Cuando se desee mostrar un elemento oculto solo debe llamar la clase mostrar y asi los elementos seran mostrados.
Este div esta oculto
Ahora ya se observa el div, se puede mostrar en pantalla. Codigo
<div class="rde-s p-5 b-negro-1 sf-4 m-5 cf-blanco mostrar">Este div se esta oculto.</div>
