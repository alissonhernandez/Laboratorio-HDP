Formularios Proporciona mensajes de retroalimentacion contextuales para las acciones típicas de los usuarios con mensajes de alerta disponibles y flexibles Ejemplo Se puede crear de manera sencila un formulario de inicio de sesion, creando un div con la clase form e ingresando las etiquetas correspondientes
Ingresa tu correo electronico aqui
<div class="form">
    <label class="form-etiqueta">Correo Electrónico </label>
    <input type="text" class="form-control" aria-placeholder="email">
    <div class="form-text"> Ingresa tu correo electronico aqui </div>
    <label class="form-etiqueta"> Contraseña </label>
    <input type="password" class="form-control" aria-placeholder="password">
    <div class="form-checkbox">
        <input type="checkbox" name="a" id="check">
        <label for="" class="form-check-label">Recordar contraseña </label>
    <button type="submit" class="btn-sm-rojo miz-5 mab-5">Enviar </button>
</div>
ComboBox o listas Se puede crear un combobox o lista con multiples opciones
<select name="" id="" class="form-Cmb-large" size="1">
        <option selected>Abre el menu </option>
        <option value="1">Opcion 1 </option>
        <option value="2">Opcion 2 </option>
        <option value="3">Opcion 3 </option>
</select>
Tambien se puede cambiar el tamaño de los elementos que se mostraran en la lista a la vez
<select name="" id="" class="form-Cmb-large" size="3">
        <option selected>Abre el menu</option>
        <option value="1">Opcion 1</option>
        <option value="2">Opcion 2</option>
        <option value="3">Opcion 3</option>
        <option value="4">Opcion 4</option>
        <option value="5">Opcion 5</option>
</select>
Existen varios tamaños para los comboBox o listas, utilizando las medidas: small, medium, large y xl
<select name="" id="" class="form-Cmb-small" size="1" >
        <option selected>Tamaño Small </option>
</select>
<select name="" id="" class="form-Cmb-medium" size="1" >
        <option selected>Tamaño Medium </option>
</select>
<select name="" id="" class="form-Cmb-large" size="1" >
        <option selected>Tamaño Large </option>
</select>
<select name="" id="" class="form-Cmb-xl" size="1" >
        <option selected>Tamaño XL </option>
</select>
Desactivado Todos los apartados del formulario se pueden desactivar, para que el usuario no pueda seleccionar o presionar algun elemento
<label class="form-etiqueta">ComboBox desactivado </label> <select name="" id="" class="form-Cmb-Xl" size="1" >
        <option selected>Abre el menu </option>
        <option value="1">Opcion 1 </option>
        <option value="2">Opcion 2 </option>
        <option value="3">Opcion 3 </option>
</select>

<label class="form-etiqueta">Boton desactivado </label>
<button disable="disabled" class="btn btn-disable-sm">Desactive </button>

<label class="form-etiqueta">TextBox Desactivado </label>
<input disabled type="text" class="form-control disable" aria-placeholder="email">
